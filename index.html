GetAuth

This is a really simple tool to get a valid Mastodon OAuth2 token for use with the Mastodon API.

Everything here happens in your browser - there is no server side component to. View source on GitHub.

Firstly, give your app a name. This name is for your internal reference only.
This name will be displayed on your 'Authorized apps' section in Mastodon, so choose something that will help you remember what this is.

Secondly, provide a list of scopes you need. See the documentation for details.
You can list as many scopes as you need - separate them with a whitespace
